# PRACTICE.md — template

Copy to your practice workspace (or let the skill create it on your first session). One line per session — the facilitator reads it to rotate katas and variations so repetition stays varied, and the takeaway column is where five exercises become one visible trend.

```markdown
# Practice log

| Date | Kata | Variation | Format | Takeaway (one phrase) |
|---|---|---|---|---|
| 2026-07-05 | module-depth | invented pairs | pattern-recognition | I over-rely on interface size; missed caller burden |
| 2026-07-12 | review-notification-preferences | cold read | review | green CI blinded me to the silent default change |

## Recurring observations

<When the same takeaway appears twice, promote it here — it's a practice target now,
and worth telling craft-coach about via your COACHING.md "trying on" list.>
```

Rules the log serves: prefer a kata and variation you haven't run recently; a kata with one variation is a kata-quality issue, not a rotation failure; the log records what *you* noticed — the facilitator never grades.
