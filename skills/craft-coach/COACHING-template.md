# COACHING.md — template

Copy to the root of a workspace where craft-coach runs, and edit honestly. This is the signed half of the asymmetric-binding contract: the coach surfaces only what you've actually signed up for.

```markdown
# Coaching contract

## Adopted — surface freely
- two-hats
- yagni
- code-smells

## Trying on — surface gently, at most once a session
- red-green-refactor
- listen-to-the-tests

## Off — never surface
- (anything you've decided isn't your practice right now)

## Session notes
- Spikes/prototypes: I'll say "this is a spike" — disciplines for long-lived code are off duty when I do.
```

Move concepts between lists as your practice changes; the file is yours, not the coach's.
